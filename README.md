# django_face-detection

## create virtual environment
```
python3 -m venv env
```

## active environment MacOS
```
source ./env/bin/activate
```

## install lib
```
pip install -r requirements.txt
```

## Run App
```
cd face_api
python manage.py runserver
```
go http://localhost:8000/
